# tiny snails

## Build Setup

```bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start
```

The final product is also hosted temporarily [here](https://tny-snls.xyz/).
